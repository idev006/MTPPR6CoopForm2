### C-02 — `04_architecture.md` และ `07_frontend.md` สะท้อน directory structure เก่า

**ไฟล์:** `04_architecture.md` (section 4.4), `07_frontend.md` (section 7.2)

**ปัญหา:** Sprint 10 ย้าย components จาก `src/components/wizard/` → `src/forms/` แต่ทั้งสองไฟล์ยังแสดง path เก่า

```
doc บอกว่า:
  components/forms/ordinary-loan/Step*.vue

ความจริง (post Sprint 10):
  src/forms/ordinary-loan/
  src/forms/shared/
  src/forms/staff/
  src/forms/emergency-loan/   ← hidden, created Sprint 10
```

**ผลกระทบเพิ่มเติมใน 07_frontend.md:**
- ยัง list composables (`useFormWizard.ts`, `useAutoSave.ts`, `useLoanApplication.ts`) แต่ Sprint 5.5 ย้าย auto-save logic เข้า `form.store.ts` โดยตรง
- `application.store.ts` ถูก list ใน stores แต่ functionality ถูกรวมเข้า `form.store.ts` แล้ว
- directory structure ใน `07_frontend.md` section 7.2 อ้างอิง `components/forms/` path ซึ่งไม่มีแล้ว
